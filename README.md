### 커밋, pr 규칙
- FEAT, FIX, CHORE, DESIGN, DELETE를 기본으로 사용 & pr은 기본적으로 아래 양식
```### 📌 관련 이슈
- Fixes #이슈번호

### 📝 작업 내용 (작업종류 - FEAT, FIX, CHORE, DESIGN, DELETE)
- [ ] **FEAT**: 신규 기능 추가
- [ ] **FIX**: 버그 수정
- [ ] **CHORE**: 빌드 업무, 패키지 매니저, 단순 오타 수정 등
- [ ] **DESIGN**: CSS 등 사용자 UI 디자인 변경
- [ ] **DELETE**: 코드 삭제

### 📸 스크린샷 (있으면)


### 📚 기타 참고 사항
- 

### 🔎 참조 인원
- @
```
